# 安装

1 拖拽 ```ProvisionDart.qlgenerator```至```QuickLook的替身```

2 打开Terminal输入```qlmanage -r```



# 使用ProvisionDart.qlgenerator安装

1 将ProvisionDart.qlgenerator放置与 /Libaray/QuickLook/下

2 打开Terminal输入```qlmanage -r```
